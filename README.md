# 🏸 Automatic Badminton Shuttle Launcher

A fully mechanical, auto-loading badminton shuttlecock launcher inspired by the click-mechanism of ballpoint pens.

## 🎯 Project Overview
This project involves the complete mechanical design, mathematical modeling, and physical manufacturing of a badminton launcher. It features a custom gear-and-rack transmission system that simultaneously compresses the firing spring and gravity-feeds a shuttlecock from a 10-20 capacity magazine.

## 🛠️ Engineering Highlights
* **Kinematic & Energy Analysis:** Modeled the spring mechanism using Simple Harmonic Motion (SHM) to optimize kinetic energy transfer and system efficiency.
* **CAD & FEA Simulation:** Designed entirely in **Autodesk Fusion 360**. Conducted Finite Element Analysis (FEA) to evaluate peak von Mises stress against the yield strength of PLA materials, ensuring a safety factor greater than 1.
* **Prototyping & Iteration:** Manufactured using FDM 3D printing (Prusa i3 MK3). Iteratively refined the cam follower profile and optimized assembly tolerances (from 0.5mm to 1.5mm) to resolve mechanical jamming.

## 📂 Repository Contents
* `CAD_Models/`: Contains universal `.STEP` and `.STL` files exported from Fusion 360 for cross-platform viewing and 3D printing.
* `Group_03_final_report.pdf`: The comprehensive engineering report detailing the mathematical models, stress calculations, and manufacturing processes.
